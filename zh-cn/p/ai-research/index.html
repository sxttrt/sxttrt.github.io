二、项目背景与目标

1. 项目背景

深度学习虽在大规模复杂任务中表现突出，但存在参数冗余、可解释性弱、依赖高算力的局限；而传统机器学习算法（如K-means、PCA、Logistic Regression）因原理透明、轻量化、易复现，在中小规模数据场景（如种子分类、图像降维）中仍具重要研究价值。

本项目以“探索传统算法的性能边界与参数影响规律”为核心，聚焦两类典型任务：

  • 分类任务：包含“种子数据集聚类”（K-means/Soft K-means）与“X型分布数据二分类”（Logistic Regression/MLP），探索算法类型、超参数对分类效果的影响；
  • 图像处理任务：基于PCA、Linear Autoencoder、Soft K-means实现图像重建，探索“保留维度/聚类数”与“重建质量”的关联。

所有算法均通过纯Python/NumPy实现，不依赖第三方机器学习库，旨在深入理解算法底层逻辑，并通过对比实验总结可复用的参数调优规律。

2. 核心目标

  • 聚类分类探索：以种子数据集为对象，探索K-means与Soft K-means在“不同聚类数、分裂合并策略、迭代数、容忍度”下的聚类效果，对比两类算法的准确率稳定性与抗局部最优能力，总结参数对聚类性能的影响规律；
  • 图像重建探索：以WWI士兵图像为处理对象，分别用PCA、Linear Autoencoder、Soft K-means进行重建，探索“保留维度（PCA/AE）、聚类数（Soft K-means）”对图像色彩还原、细节保留的影响，明确不同算法在图像处理中的优劣场景；
  • 二分类探索：针对X型分布的人工生成数据，对比线性模型（Logistic Regression）与非线性模型（MLP）的拟合能力，重点探索MLP中“隐藏层数量、神经元数、容忍度”对收敛速度、拟合精度的影响，明确非线性模型适配复杂数据的关键条件；
  • 工程化支撑：将所有算法模块化封装，配套实验可视化脚本（准确率曲线、重建效果对比图），确保实验过程可复现、结果可直观解读，为后续传统算法应用提供参考模板。

三、核心任务实现

任务一：K-means与Soft K-means聚类分类（种子数据集探索）

1. 探索方向与算法设计

本次探索聚焦“算法类型（K-means/Soft K-means）、超参数（聚类数、分裂合并、迭代数、容忍度）对聚类准确率的影响”，为匹配探索目标，对两类算法进行针对性设计：

  • K-means设计（服务探索目标）：

    • 核心逻辑：通过最小化簇内平方误差
      $$J=\sum_{i=1}^{k} \sum_{x \in C_{i}}\left| x-\mu_{i}\right| ^{2}$$
      （$C_i$为第$i$个簇，$\mu_i$为簇中心）实现聚类；
    • 关键优化：针对“分裂合并策略对聚类稳定性的影响”，设计“拆分最大距离簇+合并最近簇”的动态调整方案（替代原阈值法，避免聚类数爆炸），便于探索策略有效性；
    • 标签校准：通过“簇内真实标签占比最高者修正预测标签”，确保准确率计算可用于横向对比。

  • Soft K-means设计（服务探索目标）：

    • 核心逻辑：引入概率权重 $w_{ij}$（数据$x_i$属于簇$j$的概率），目标函数优化为
      $$J=\sum_{i=1}^{n} \sum_{j=1}^{k} w_{ij}\left| x_{i}-\mu_{j}\right| ^{2}$$
      探索“概率化聚类”对准确率稳定性的提升；
    • 关键优化：针对“初始质心对收敛的影响”，设计“按距离平方概率分布选质心”的策略（先随机选1个质心，后续质心优先选择与已有质心距离远的样本），对比K-means随机初始化的效果；
    • 参数固定：β=0.7（经预实验确定，避免参数过多干扰探索结论）。

2. 探索实验与结果分析

基于种子数据集（含3类种子，共210个样本），设计4组对比实验，探索核心参数影响：

（1）容忍度影响探索（k=3，无分裂合并）
模型 最大迭代 容忍度 准确率
K-means 300 1e-3 79.52%
K-means 300 1e-4 79.52%
K-means 300 1e-5 79.52%
Soft K-means 300 1e-3 89.52%
Soft K-means 300 1e-4 89.52%
Soft K-means 300 1e-5 89.52%

探索结论：容忍度在$1e-3~1e-5$范围内，两类算法准确率均无变化，说明此参数对种子数据集聚类结果影响极小，后续实验可固定容忍度=1e-5（兼顾迭代效率与精度）。

（2）迭代数影响探索（k=3，容忍度=1e-5）
模型 最大迭代 容忍度 准确率
K-means 100 1e-5 73.33%
K-means 200 1e-5 89.05%
K-means 300 1e-5 79.52%
K-means 400 1e-5 76.19%
K-means 500 1e-5 76.67%
Soft K-means 50 1e-5 89.52%
Soft K-means 100 1e-5 89.52%
Soft K-means 200 1e-5 89.52%
Soft K-means 300 1e-5 89.52%

探索结论：

  • K-means：迭代数=200时准确率最高（89.05%），超200次后准确率下降（可能过拟合）；
  • Soft K-means：迭代数≥50时准确率稳定在89.52%，无明显波动，说明其对迭代数敏感度更低，收敛更稳定。
（3）聚类数与分裂合并策略影响探索（迭代数=200，容忍度=1e-5）
模型 聚类数 分裂合并 准确率
K-means 3 False 89.05%
K-means 10 False 86.19%
K-means 10 True 86.19%
Soft K-means 3 False 89.52%
Soft K-means 10 False 90.00%
Soft K-means 10 True 89.05%

探索结论：

  • 聚类数影响：K-means聚类数从3增至10时准确率下降3%，Soft K-means则提升0.48%，说明Soft K-means更适配多聚类场景；
  • 分裂合并策略影响：仅Soft K-means启用策略后准确率下降0.95%（种子数据簇分布紧凑，策略反而干扰聚类），K-means无变化，说明策略需结合数据分布使用。

任务二：PCA、Linear Autoencoder与Soft K-means图像重建探索

1. 探索方向与算法实现

本次探索聚焦“算法类型、关键参数（保留维度/聚类数）对图像重建质量的影响”，选择WWI士兵图像（256×256像素，低色彩复杂度，避免RGB通道干扰）为处理对象，三类算法实现均服务于探索目标：

  • PCA实现（探索“保留维度”影响）：
    步骤：数据中心化→协方差矩阵计算
    $$C=\frac{1}{n} X_{centered }^{T} X_{centered }$$
    →SVD分解
    $$C=U \sum V^{T}$$
    →投影到前$k$主成分（$$X_{projected }=X_{centered } V[:,:k]$$）；
    重建：通过
    $$X_{recovered }=X_{projected } V[:,:k]^{T}+\mu$$
    还原图像，重点探索$k=1/2/3$时的效果差异。

  • Linear Autoencoder实现（探索“编码维度”影响）：
    结构：输入层（图像像素）→编码层（$$E=X \cdot W_e$$）→解码层（$$D=E \cdot W_d$$）（无激活函数，纯线性变换）；
    训练：MSE损失
    $$loss =\frac{1}{m} \sum_{i=1}^{m}(D-X)^{2}$$
    ，梯度裁剪避免爆炸（初始权重×0.00001防对称问题），重点探索编码维度=1/2/3时的重建误差。

  • Soft K-means实现（探索“聚类数”影响）：
    逻辑：将图像像素视为样本，按聚类数$k$分配权重，用簇中心像素值替换原像素实现重建，重点探索$k=1/3/5/7/9$时的色彩与细节还原效果。

2. 探索实验与结果分析

（1）PCA与Linear Autoencoder重建对比（按维度）

图3：Linear Autoencoder重建效果（左→右：原图、encoding_size=1/2/3）

探索结论：

  • 维度=1时：两类算法均输出灰度图（丢失色彩信息），Linear Autoencoder重建误差（0.015）略低于PCA（0.018）；
  • 维度=2时：Linear Autoencoder已呈现明显色彩（误差0.008），PCA仅轻微色彩（误差0.012），说明AE线性变换对特征保留更优；

（2）Soft K-means重建对比（按聚类数）

探索结论：

  • $k=7~9$时：色彩逐渐丰富，$k=9$时覆盖所有像素主色，边缘细节（如帽子、衣领）基本还原，说明聚类数需匹配图像色彩复杂度（此图$k=9$为最优）。

任务三：Logistic Regression与MLP二分类探索（X型分布数据）

1. 探索方向与算法实现

本次探索聚焦“模型类型（线性/非线性）、MLP超参数（隐藏层/神经元/容忍度）对二分类效果的影响”，基于人工生成的X型分布数据（2维特征，两类样本呈交叉分布）设计实验：

  • Logistic Regression实现（线性模型基准）：
    激活函数：Sigmoid
    $$\sigma(z)=\frac{1}{1+e^{-(w^T x + w_0)}}$$
    ；
    训练：交叉熵损失
    $$loss =-\frac{1}{m} \sum_{i=1}^{m}[y_i log(h_i)+(1-y_i)log(1-h_i)]$$
    ，梯度下降更新权重（$lr=0.01$，迭代=10000），探索线性模型对非线性数据的拟合局限。

  • MLP实现（非线性模型探索）：
    结构：输入层（2维特征）→隐藏层（1-6层）→输出层（Sigmoid激活），支持自定义神经元数；
    优化：梯度裁剪控制梯度范围，重点探索“隐藏层数量（1/2/6）、神经元数（2/4/8/20）、容忍度（1e-5~1e-8）”对收敛与准确率的影响；
    停止条件：当两次迭代损失差绝对值＜容忍度时停止，记录收敛次数与最终准确率。

2. 探索实验与结果分析

（1）线性vs非线性模型对比（基础探索）
模型 隐藏层/神经元 容忍度 准确率 收敛次数
Logistic Regression - 1e-5 59.60% 10000（未收敛）
MLP 1/8 1e-7 100.00% 8320

探索结论：X型分布数据（非线性边界）无法被Logistic Regression有效拟合（准确率仅59.6%），而MLP（1隐藏层+8神经元）可完全拟合，说明非线性模型对复杂数据的适配优势。

（2）MLP超参数影响探索
① 隐藏层与神经元数影响（容忍度=1e-7）
模型配置（隐藏层/神经元） 收敛次数 最终损失 损失变化趋势（对应图5曲线） 准确率
1隐藏层+2神经元 1215 0.1523 快速收敛后停滞，损失居高不下 78.67%
1隐藏层+4神经元 8546 0.0253 缓慢收敛，损失逐步下降 98.67%
1隐藏层+8神经元 8320 0.0132 收敛速度中等，损失逼近最小值 100.00%
2隐藏层+3+6神经元 7264 0.0050 收敛平稳，损失最低且无波动 99.67%
6隐藏层+2神经元 17 0.1628 瞬间收敛（未充分训练），损失高 77.00%

探索结论：

  • 1隐藏层：神经元≥8时可完全拟合，神经元过少（2个）则拟合不足；
  • 2隐藏层：3+6神经元即可实现99.67%准确率，无需过多神经元；
  • 6隐藏层：即使神经元足够，仍因层数过多陷入局部最优（准确率77%），说明隐藏层并非越多越好。
② 容忍度影响（2隐藏层+3/6神经元）
容忍度 收敛次数 损失 准确率
1e-5 149 0.2489 53.67%
1e-6 43 0.2493 59.00%
1e-7 7264 0.0050 99.67%
1e-8 8120 0.0048 99.50%

探索结论：

  • 容忍度≥1e-6时：损失早停（未充分拟合），准确率仅53.67%~59%；
  • 容忍度=1e-7~1e-8时：损失充分收敛，准确率≥99.5%，但1e-8时因过度迭代出现轻微过拟合（准确率微降0.17%），最优容忍度为1e-7。

2. 核心探索成果（规律总结）

探索方向 关键结论
聚类分类 1. Soft K-means比K-means更稳定（准确率波动≤0.5% vs 3%），多聚类场景（k=10）更优；
2. 分裂合并策略仅适用于簇分布松散数据，种子数据（紧凑分布）启用后准确率下降0.95%；
3. 容忍度（1e-3~1e-5）对聚类结果影响极小，迭代数需匹配算法（K-means=200，Soft K-means≥50）
图像重建 1. PCA需保留3维（RGB通道）才能完全还原图像，Linear Autoencoder 2维即可实现低误差（0.008）；
2. Soft K-means聚类数需匹配图像色彩复杂度，WWI士兵图k=9时细节还原最佳；
3. Linear Autoencoder对初始权重敏感，随机权重易导致重建失败
二分类 1. 非线性数据（X型分布）需用MLP（非线性模型），Logistic Regression（线性）准确率仅59.6%；
2. MLP最优配置：1~2隐藏层+8/3+6神经元，容忍度=1e-7（平衡收敛与过拟合）；
3. 隐藏层≥6时易陷入局部最优，准确率骤降22%+
